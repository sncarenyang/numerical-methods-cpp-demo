# Numerical Methods C++ Demo

A lightweight **C++17** demo implementing **Newton-Raphson** root-finding and **Trapezoidal Rule** numerical integration. Perfect for computational physics/engineering portfolios.

## Features
- Solves \\( x^2 = 2 \\) → √2 ≈ **1.414213**
- Integrates sin(x) from 0 to π ≈ **2.0** (n=100)
